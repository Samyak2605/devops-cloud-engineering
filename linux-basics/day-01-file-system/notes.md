# Day 1: Linux File System Basics

## What is Linux?
Linux is an open-source operating system widely used in servers, cloud, and DevOps environments.

## Important Directories
- / : Root directory
- /home : User home directories
- /etc : Configuration files
- /var : Logs and variable data
- /bin : Essential binaries

## Absolute vs Relative Path
- Absolute path starts from root (/home/user)
- Relative path starts from current directory

## Why Linux for DevOps?
- Most servers run Linux
- Automation and scripting are easier
- Essential for Docker, Kubernetes, and Cloud
